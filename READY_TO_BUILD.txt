================================================================================
  üêõ TROUBLESHOOTING
================================================================================

Build Fails:
  - Increase Docker memory to 4GB+
  - Try: docker build --no-cache -t rehau-bridge:v5.0.0 .

Port 3000 In Use:
  - Use different port: docker run -p 3001:3000 ...

Auth Fails:
  - Check REHAU_EMAIL and REHAU_PASSWORD in .env
  - Verify POP3 settings if 2FA enabled
  - Check logs: docker logs rehau-test | grep "üîê\|‚ùå"

MQTT Not Connecting:
  - Verify MQTT_HOST and MQTT_PORT
  - Check MQTT broker is running
  - Test: mosquitto_pub -h localhost -t test -m "hello"

High Memory:
  - Normal with Playwright (300-500 MB)
  - Browser closes after 5 min idle
  - Check: docker stats rehau-test
